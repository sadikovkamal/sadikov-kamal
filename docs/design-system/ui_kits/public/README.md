# Provia public UI kit

Two screens that sit outside the admin login wall:

| File | Screen |
|---|---|
| `index.html` | Landing (`/`) — the "tez orada" coming-soon page with brand mark and a single login affordance. |
| `login.html` | Login (`/login`) — admin sign-in card with error + rate-limit states. |

Both share `../_shared.css` for tokens and primitives. Link from one to the other via the brand mark and the "Kirish →" link.

The landing leans into the slogan **"Isbotga yo'l"** as the page title — short, confident, no marketing copy. Tagline restates the product description from the spec.
