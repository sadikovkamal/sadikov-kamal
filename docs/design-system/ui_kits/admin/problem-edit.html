Faraz qilaylik, x, y, z — musbat butun sonlar va ular quyidagi shartni qanoatlantiradilar:

x² + y² = z²

Quyidagilarni isbotlang:

  • Agar gcd(x, y, z) = 1 bo'lsa, u holda x va y dan biri juft son bo'lishi shart.
  • Bunday uchliklar soni cheksiz ko'p.
